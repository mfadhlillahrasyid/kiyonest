Kiyonest Website

Overview

Kiyonest merupakan Produk minuman kesehatan yang terbuat dari sarang walet asli, diproses secara natural tanpa bahan pengawet buatan. Kiyonest mendevelop sebuah landing page yang bertujuan untuk informasi seputar produk dengan kualitas terbaik dan memiliki cita rasa tinggi. Dengan adanya Landing page ini, Semoga dapat membantu para Konsumen untuk lebih bijak dalam memilih minuman sarang walet dengan kualitas nomor satu.

Feature

  • Landing Page A user-friendly interface that provides an overview of the product and its features
  • Demo Version Built using HTML, CSS, JavaScript, and Bootstrap UI, this version allows users to try out the product's basic features
  • Pro Version Built using Laravel, this version includes a comprehensive Admin Panel and additional features

Getting Started

Demo Version

To get started with the demo version of Kiyonest, simply open the landing page in your web browser. You can explore the available features and get a feel for how the product works.

Pro Version

To get started with the pro version of Kiyonest, follow these steps:

  • Installation: Follow the installation instructions provided with the Laravel package
  • Configuration: Configure the Admin Panel by following the instructions provided in the Laravel documentation
  • Launch: Launch the application and start using the pro version of Kiyonest

Usage

Demo Version

To use the demo version of Kiyonest, simply navigate through the landing page and explore the available features.

Pro Version

To use the pro version of Kiyonest, log in to the Admin Panel and start using the product's features. For more information on how to use the pro version, please refer to the Laravel documentation and the Kiyonest user guide.
